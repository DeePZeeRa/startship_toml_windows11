# Installation:
1. Follow and Install startship using installation guide: `https://starship.rs/installing/`.
2. Install Nerd font from `https://www.nerdfonts.com/font-downloads`. I am using **JetBrainsMono Nerd Font**. You can change the font in windows terminal/powershell settings.
3. To change font, Click on the **Down arrow (⌄)** at the top near tab area -> **Default/your desired shell or terminal** -> **appearance**.
4. Place the **starship.toml** file inside: `C:\Users\<YourUser>\.config\`
